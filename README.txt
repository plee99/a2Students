Assingment: Displaying and Modifying Database of Students 
            using PostgreSQL and SpringBootApplication


FEATURES: 
            1. Users can freely delete, add, and update students as they wish

            2. Students are displayed as rectangles with color, width, height associated with student attributes. GPA and Id are displayed as well

            3. Additional attribute "int age" implemented and displayed in the top left corner of student rectangles

            4. Additional attribute "String email" implemented and can be seen by hovering over email icon displayed at top right corner of student rectangles

            5. Out of bounds error checking for weight and height 

            6. CSS effects added for usability and aesthetics
